---
name: web-search
description: >
  Search the web for current events, news, factual queries, and real-time
  information using OpenCode's built-in WebSearch tool. Use when the user
  asks about anything that requires up-to-date or external knowledge.
---

When the user asks for information that may be time-sensitive or outside
your training data:

1. Use the `WebSearch` tool to search the web for the requested information.
2. Read relevant results and synthesise a concise answer.
3. Respond in 1-3 sentences in the same language as the user.
4. Never make up facts or URLs — always verify via search.
